Built another wheel again! I completley rewrote an existing project named BroadCast Channel from scratch. It syncs content from a Telegram channel into a searchable, multilingual static microblog, with support for local media mirroring, RSS, and tag pages.…
Surprisingly, after tightening nearly every corner of the stack from static generation and CDN delivery to image optimization and caching; this actually turned out to be one of the fastest sites I’ve built so far. Seeing the performance audits and SEO scores come out this strong was genuinely satisfying; it’s always rewarding when careful taking different measures; in spite of taking wiring multiple approach routes that fnally translates directly into a clean, lightning-fast user experience is even possible via the infamous and notorious NextJS React stack. #dev
